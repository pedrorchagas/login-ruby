# Sistema de login 
Um simples sistema de login feito em Ruby

## Porque eu fiz?
Uma ótima forma de aprender é praticando, por isso eu desenvolvi esse pequeno projeto para aprender tecnologias e práticas que todos sistemas de login deve ter.

## Funcionalidades
 - Login
 - Registro
 - Cache de sessões

## Tecnologias
 - Redis
 - MySql
 - Ruby
 - Html e CSS puros
 - Sinatra (gem do Ruby)
